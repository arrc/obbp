# Online Blood Bank Project (Obbp). [live](http://obbp.arrc.in)

### Admin section
  - approve user
  - process blood request
  - search users
  - manage camp schedules

### Public
  - search donors.
  - request blood.
  - view upcoming camps details.
  - Create and edit own profile.
